Features

Everything you need for full Mac automation

Web Automation

8 Playwright tools: navigate, click, type, extract, screenshot, wait, scroll, eval. Direct DOM access — no pixel guessing.

Model Router

Auto-routes each prompt to Haiku (simple), Sonnet (medium), or Opus (complex). Customizable rules. ~90% token savings.

Native macOS

PyAutoGUI mouse/keyboard control + AppleScript for native apps. Open Spotify, control Finder, take screenshots.

Telegram Bot

Control your Mac from anywhere. Natural language, persistent memory, self-learning with fuzzy matching.

Web Dashboard

FastAPI-powered dashboard at localhost:8080. REST API, NLP assistant mode, real-time logs. Accessible via Tailscale.

Smart Autonomy

Three modes: full (no confirmation), checkpoint (pause on destructive), plan_first (show plan before acting).
