Perguntas Frequentes

Respostas rápidas sobre InfiniteClaud

Como funciona o Model Router? ▼
O Model Router analisa cada prompt e roteia automaticamente para o modelo ideal: Haiku para tarefas simples, Sonnet para automação multi-step, e Opus para raciocínio profundo. Resultado: 90% menos tokens por sessão.
Quais são as 3 interfaces de acesso? ▼
Claude Code (CLI interativo), Dashboard Web (localhost:8080 ou remoto via Tailscale), e Telegram Bot (para controle remoto completo com memória persistente).
InfiniteClaud é melhor que Computer Use? ▼
Sim! 50x mais rápido (<100ms vs 2-5s), 90% menos tokens, acesso direto ao DOM/APIs nativas, sem dependência de screenshots, e com inteligência de roteamento de modelos.
Como instalar? ▼
Um comando: claude plugin install --from github Silvakazu-cell/infiniteclaud. Ative com /InfiniteClaud e comece a usar imediatamente.
