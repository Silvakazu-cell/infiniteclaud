Perguntas Frequentes

O que é um MCP Server? +
MCP (Model Context Protocol) é o padrão da Anthropic para ferramentas de IA. InfiniteClaud implementa 20 ferramentas nativas como Web, Filesystem, Native macOS e System, permitindo que Claude acesse e manipule o seu computador de forma segura e eficiente.
Como o Model Router economiza tokens? +
O Model Router analisa a complexidade de cada tarefa. Tarefas simples usam Haiku (super rápido), médias usam Sonnet (balanceado), e apenas as mais complexas usam Opus. Isso resulta em ~90% de economia de tokens comparado a usar Opus em tudo.
Preciso de conhecimento técnico para usar? +
O básico (instalação) requer conhecimento de terminal/command line. Mas a interface Web e Telegram Bot são intuítivas e não requerem nenhum conhecimento técnico para usar. O SDK oferece documentação completa com exemplos prontos.
Quais são os requisitos de sistema? +
Python 3.11+, qualquer SO (Windows, macOS, Linux). As ferramentas nativas do macOS (AppleScript, mouse, teclado) funcionam apenas em macOS. Web e Filesystem funcionam em todos os sistemas operacionais.
Posso usar com minha API key existente? +
Sim! InfiniteClaud funciona com sua chave API Anthropic existente. Você não precisa mudar nada, apenas adicione a chave no arquivo .env. O plugin reutiliza sua assinatura existente.
