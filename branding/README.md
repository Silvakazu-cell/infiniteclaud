## 🎯 Quick Start

### 1. Add Favicon to Your Website
```html

<link rel="apple-touch-icon" href="branding/favicon-256x256.png">
```


## 🚀 Best Practices

✅ **DO:**
- Use logo on dark backgrounds
- Maintain aspect ratio when resizing
- Preserve whitespace around logo
- Use full banner for social sharing
- Update favicon after deployment

❌ **DON'T:**
- Stretch or distort logo
- Change colors without permission
- Use low-res versions
- Crop or modify composition
- Remove credit/branding
